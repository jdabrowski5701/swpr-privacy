# Privacy Policy Last updated: May 2026 ## 1. Introduction SWPR Export is an academic research application developed as part of a master's thesis project focused on movement pattern detection using smartwatch and Huawei Health data. The application is intended for research and educational purposes only. ## 2. Data Collection The application may access and process the following data from Huawei Health, subject to user authorization: * Step count data * Activity records * Heart rate data * Distance and calorie data * Sleep-related data * SpO2 and stress data * Sport activity information * Historical fitness and health data The application only accesses data explicitly authorized by the user through Huawei Health and Huawei Health Service Kit. ## 3. Purpose of Data Processing The collected data is used exclusively for: * movement pattern analysis, * time-series analysis, * machine learning experiments, * academic research related to the master's thesis project. The application is not intended for medical diagnosis, treatment, or healthcare decision-making. ## 4. Data Storage The application stores exported data locally on the user's device unless the user explicitly transfers the exported files elsewhere. No health data is intentionally shared with third parties. ## 5. Third-Party Services The application uses Huawei Health and Huawei Health Service Kit APIs provided by Huawei Technologies Co., Ltd. Use of these services may also be subject to Huawei's privacy policies and terms. ## 6. User Consent By using the application and granting permissions through Huawei Health authorization screens, the user consents to the processing of authorized health and activity data for research purposes. ## 7. Data Security Reasonable efforts are made to protect locally processed data. However, the application is provided for research purposes without guarantees regarding security, availability, or reliability. ## 8.
